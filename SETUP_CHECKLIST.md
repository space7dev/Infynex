# Project Setup Checklist

Use this checklist to verify your project is set up correctly.

## Initial Setup

- [ ] Node.js 18+ installed
- [ ] Dependencies installed (`npm install`)
- [ ] `.env.local` file created from `.env.example`
- [ ] VS Code extensions installed (see `.vscode/extensions.json`)

## Configuration

- [ ] Site name updated in `lib/config.ts`
- [ ] Contact information updated in `lib/config.ts`
- [ ] Social media links updated in `lib/config.ts`
- [ ] Brand colors customized in `tailwind.config.ts`
- [ ] Logo replaced in `public/logo.svg`
- [ ] Favicon updated in `public/favicon.png`

## Content

- [ ] Homepage content reviewed and updated
- [ ] Service pages content customized
- [ ] About page information updated
- [ ] Portfolio/case studies added
- [ ] Testimonials updated with real data
- [ ] Contact form tested
- [ ] All placeholder text replaced

## Development

- [ ] Development server runs (`npm run dev`)
- [ ] All pages accessible and load correctly
- [ ] No console errors or warnings
- [ ] ESLint passes (`npm run lint`)
- [ ] Code formatted (`npm run format`)
- [ ] TypeScript compiles without errors

## Features

- [ ] Navigation menu works on desktop
- [ ] Mobile hamburger menu works
- [ ] All internal links functional
- [ ] Contact form validation works
- [ ] Form submission configured
- [ ] Responsive on mobile, tablet, desktop
- [ ] Smooth animations and transitions
- [ ] CTA buttons work correctly

## SEO & Performance

- [ ] Meta tags on all pages
- [ ] OpenGraph tags configured
- [ ] Sitemap generates correctly
- [ ] Robots.txt configured
- [ ] Images optimized with next/image
- [ ] Performance tested (Lighthouse 90+)
- [ ] Accessibility tested (Lighthouse 90+)

## Production Build

- [ ] Production build succeeds (`npm run build`)
- [ ] No build errors or warnings
- [ ] Production server runs (`npm start`)
- [ ] All pages work in production mode
- [ ] Environment variables configured

## Deployment

- [ ] Hosting platform chosen
- [ ] Environment variables set on hosting
- [ ] Custom domain connected (if applicable)
- [ ] SSL certificate active
- [ ] Deployment successful
- [ ] Live site tested and functional

## Post-Launch

- [ ] Google Analytics added (optional)
- [ ] Error tracking configured (optional)
- [ ] Contact form emails received
- [ ] Social media links verified
- [ ] Mobile testing on real devices
- [ ] Cross-browser testing complete
- [ ] Performance monitoring setup (optional)

## Documentation

- [ ] README.md reviewed
- [ ] Team members familiar with codebase
- [ ] Deployment process documented
- [ ] Maintenance plan established

## Optional Enhancements

- [ ] Dark mode

- [ ] Live chat integration

- [ ] Privacy Policy page
