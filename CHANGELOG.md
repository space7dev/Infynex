# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2026-02-05

### Added
- Initial release of Infynex Solutions website
- Next.js 14 with App Router architecture
- TypeScript support throughout the project
- Tailwind CSS with custom design system
- 12 main pages (Home, Mobile, Blockchain, AI, Products, Hire Developers, Portfolio, Testimonials, About, Contact, Case Studies, Rescue Project)
- Responsive navigation with mobile hamburger menu
- Reusable components (Hero, ServiceCard, TestimonialCard, CaseStudyCard, ContactForm)
- Contact form with React Hook Form and Zod validation
- SEO optimization (meta tags, OpenGraph, sitemap.xml, robots.txt)
- PWA manifest for installability
- ESLint and Prettier configuration
- Global layout with Navbar and Footer
- Custom Tailwind utilities and components
- Accessibility features (ARIA labels, keyboard navigation)
- Performance optimizations (image optimization, lazy loading)

### Features
